# Applicazioni web {#web-applications}

Le applicazioni Web si integrano con i rollout di esperienza chiamando l&#39;**API di funzionalità V3** per recuperare i flag di funzionalità e inserire la logica condizionale nel codice dell&#39;applicazione.

Consulta **GET Feature API V3** nella sezione Feature API di questa guida per il riferimento API completo, inclusi i parametri di richiesta, il formato di risposta e le chiamate di esempio.
