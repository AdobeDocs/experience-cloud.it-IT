Questa sezione fornisce le linee guida sull’integrazione per tutti i client di Experience Rollout, organizzati per tipo di applicazione.

Seleziona la guida corrispondente al tipo di applicazione:

2. [Applicazioni desktop](desktop-applications.md) — Integrazione diretta API REST per le app desktop
3. [Applicazioni mobili](mobile-applications.md) — Integrazione REST API per le app mobili
4. [Applicazioni Web](web-applications.md) — Integrazione REST API per le proprietà Web
5. [Servizi Web](web-services.md): integrazione lato server di SDK per i servizi back-end

## Tipi di applicazioni in breve {#overview}

| Tipo di applicazione | Integrazione consigliata |
|---|---|
| **Servizio Web / backend** | Java SDK o Node.js SDK |
| **Applicazione Web** | API REST — API delle funzioni V3 |
| **Applicazione mobile** | API REST — API delle funzioni V3 |
| **Applicazione desktop** | API REST — API per funzioni V2 |
